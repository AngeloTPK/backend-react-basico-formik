# backend-react-basico-formik
Um backend simples para uma apresentação da lib formik, acho que vai ter apenas as rotas de user e produto e os schema e conexão db com prisma 

---

## DB name: formik
Não esquecer de criar um database no workbench com o nome: formik

---

## Prisma commands para funcionar
```bash
 npx prisma generate

 npx prisma migrate dev

```

---

## TSX
Um pacote que faz com que o node entenda arquivos .ts
```bash
npm install -D tsx
```
